# 📊 HR Attrition Analysis Dashboard

> **An Interactive HR Analytics Dashboard built entirely in Microsoft
> Excel using Power Query, Power Pivot, DAX, Pivot Tables, and
> Interactive Visualizations.**

## 🚀 Project Overview

This project showcases a complete **HR Attrition Analysis Dashboard**
developed in **Microsoft Excel**. It transforms raw HR data into
interactive business insights using modern Excel analytics features.

The dashboard helps analyze employee attrition, workforce satisfaction,
performance, tenure, and overtime while allowing users to explore data
dynamically through slicers and timelines.

------------------------------------------------------------------------

## 🎯 Business Objectives

-   Analyze employee attrition across departments.
-   Monitor satisfaction metrics and workforce performance.
-   Identify overtime and attrition trends.
-   Build an interactive dashboard for HR decision-making.
-   Demonstrate end-to-end Excel analytics skills.
